# Microservice Payment Failure Handling

This Python example demonstrates basic failure handling in a microservice context. It simulates a user service attempting a payment via a payment gateway, showcasing how to gracefully handle potential connection errors from the external service.

## Language

`python`

## How to Run

Run this script using Python: `python main.py`
